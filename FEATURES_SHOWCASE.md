## 🔌 API Endpoints

### Authentication

```
POST /auth/register
├─ Input: { email, password }
└─ Output: User object

POST /auth/login
├─ Input: { email, password }
└─ Output: { access_token, refresh_token }

GET /auth/me
├─ Headers: Authorization: Bearer <token>
└─ Output: User object
```

### Links

```
POST /links/
├─ Headers: Authorization: Bearer <token>
├─ Input: { original_url, custom_slug?, expires_at? }
└─ Output: Link object with short_url

GET /links/
├─ Headers: Authorization: Bearer <token>
├─ Query: ?skip=0&limit=100
└─ Output: Array of Link objects

DELETE /links/{slug}
├─ Headers: Authorization: Bearer <token>
└─ Output: 204 No Content

GET /{slug}
└─ Output: 307 Redirect to original URL
```

### Analytics

```
GET /analytics/{slug}
├─ Headers: Authorization: Bearer <token>
├─ Query: ?days=30
└─ Output: {
    total_clicks,
    daily_clicks[],
    devices[],
    browsers[],
    sources[],
    top_locations[]
  }
```
